意思

為輸入檔案計算的 MD5 總和檢查碼與提供給 API 的總和檢查碼不相符。

這個問題有各種可能的原因。最有可能的是：

  • 檔案在傳輸過程中已損毀。
  • 提供給 API 的總和檢查碼不正確。

因應措施

請檢查以確定您提供給 API 的總和檢查碼符合磁碟上檔案的總和檢查碼。

如果他們 別 匹配，請使用正確的校驗和進行新的 API 調用。

如果校驗和 做 匹配，文件可能在傳輸過程中已損壞。請重新提交工作。

在任何一種情況下，如果重試失敗，請與我們聯繫。
