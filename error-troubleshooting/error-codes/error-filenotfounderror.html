意思

我們試圖在指定的地址下載一個文件，但沒有文件存在。我們嘗試從您的 API 請求中的「輸入」設置指定的 URL 下載。此錯誤可能意味著輸入選項中存在錯字，或者文件已被刪除，或重定向循環或其他內容。

在技術上，這可能意味著一些事情。在 HTTP 的情況下，這意味著當我們嘗試下載文件時，我們得到了一個 > = 300 的 HTTP 響應代碼，除了 403，這是一個不同的錯誤。使用 FTP 和 FTPS 這意味著我們收到了 FTP 550 響應代碼，表示該文件不存在或無法使用。通過其餘的協議，這只是意味著沒有找到該文件。

因應措施

確保該文件存在於給定的 URL，並且可以通過我們連接為用戶訪問。如果該位置不存在，請先移動檔案再試一次。如果該位置確實存在，請檢查檔案的權限，以及指定的使用者是否可以存取檔案路徑。
