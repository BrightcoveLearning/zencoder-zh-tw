意思

我們試圖在指定的地址下載一個文件，但不是視頻或音頻文件，我們得到了別的東西，比如 HTML 或 XML 文檔。這可能類似於找不到文件的錯誤頁面，但這些頁面通常會帶有 404 響應代碼（或其他一些錯誤代碼）返回，並且只有在 HTTP 響應代碼為 2XX 時才會發生此錯誤，表明該網站認為這是一個成功的請求。所以它可能是一個錯字，錯誤的輸入 URL 或配置錯誤的錯誤頁面。

重試這個工作？

出現此錯誤的作業不應該重試。了解有關重試作業的更多信息。
