意思

我們試圖將文件上傳到指定的地址，但出現權限錯誤。如果這是S3 URL，則可能需要授予對Zencoder的讀取訪問權限該存儲桶或文件。最簡單的方法是使用 S3 客戶端，如亞馬遜的 AWS 控制台。

在技術上，這意味著我們從服務器獲得了 403 拒絕訪問響應。

因應措施

確保給定 URL 的文件的權限允許 Zencoder 訪問。如果文件在S3上，則可能需要授予對Zencoder的讀取訪問權限。
