意思

此錯誤表示 CDN 已拒絕 RTMP 資料流，因為該資料流名稱已被另一個資料流使用中。換句話說，兩個 RTMP 編碼器試圖推送到相同的 CDN 入口點，第二個被拒絕，因為第一個編碼器已經走了。

更多資訊

這可能意味著：

  • 另一個 ZenCoder 工作已經串流到這個 CDN 串流名稱
  • CDN 資料流名稱正在由 Zencoder 以外的其他系統使用
  • 由於某種原因，以前的工作無法斷開與 CDN

若要避免此問題，您可以使用不同的 CDN 資料流名稱，或使用相同的資料流名稱終止任何先前的作業。如果工作因某種原因無法終止，聯繫支持帶有職位編號，我們的工程團隊將進行調查。

重試這個工作？

出現此錯誤的作業不應該重試。了解有關重試作業的更多信息。
