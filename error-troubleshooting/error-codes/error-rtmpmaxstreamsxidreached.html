意思

這個錯誤表示 CDN 已拒絕 RTMP 資料流，因為有太多的資料流使用相同的資料流識別碼。因此，超過每個 ID 的最大流數。

更多資訊

這可能意味著：

  • 在 CDN 中配置的每個 ID 的最大流太低，無法接受所有請求的流
  • 其他 ZenCoder 作業已經串流到這個 CDN 和串流 ID
  • CDN 資料流識別碼正在由 ZenCoder 以外的其他系統使用
  • CDN 可能有 '殭屍' 流，其中先前的流未正確斷開連接，並且仍在計算此 ID 的最大流限制

若要避免這個問題，您可以停止其他具有串流使用此資料流識別碼的資料流的工作，或使用其他資料流識別碼 (如果有的話)。有關更多信息，您可以聯繫支持帶有職位編號，我們的工程團隊將進行調查。

重試這個工作？

出現此錯誤的作業不應該重試。了解有關重試作業的更多信息。
