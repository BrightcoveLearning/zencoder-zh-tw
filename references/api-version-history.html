概述

您可以通過應用程序用於發佈到Zencoder的URL來指定要使用的API版本。https://app.zencoder.com/api/v2/使用V2。如果沒有指定版本，請求將使用邊緣 API，其中事情可以在沒有警告的情況下更改，因此只有在正式可用之前嘗試新功能時才使用此功能，而不是在生產中。

API 版本將會隨著新版本上線而關閉，我們會在版本關閉之前至少 6 個月通知，以便您有時間遷移應用程式。
