概述：編碼器

Zencoder 是一個強大的平台，用於轉碼隨選視頻。

簡介

Zencoder 為視頻和音頻文件提供快速可靠的轉碼服務。它可以為網絡和各種各樣的設備產生輸出，並且可以輕鬆地與您的 CMS 和其他應用整合。

使用者介面

贊編碼器主要是一個 API，但有一個 UI，其中包括：

  • 帳戶設定
  • 顯示最近工作結果的儀表板
  • 一個 API 請求生成器，允許您開始使用 API 而無需編寫代碼

API

您可以使用 API 執行多種功能，其中最重要的是提交和管理隨選視訊的編碼工作。

有關API端點的完整指南，請參見API參考。

Zencoder 的強大功能在於它提供的靈活性，讓您創造出您想要的輸出類型。這也是一些複雜性的來源，因為有許多編碼設置需要理解。一個開始探索這些的好地方是以下內容的摘要編碼設定，其中包含一些鏈接，可帶您進入帶有示例代碼的完整描述。
