獲取 API 密鑰

所有對 Zencoder API 的請求都必須使用 API 金鑰進行驗證。本主題說明 Zencoder 的不同類型的 API 金鑰，以及如何取得這些金鑰。

取得 API 金鑰

要請求API密鑰，請轉到Zencoder網站。如果您已註冊為使用者，請登入。如果沒有，請註冊為新用戶-在註冊為用戶之前，您無法請求 API 密鑰。

一旦您登錄，請轉到 API 部分：

在 API 部分，你會看到要求不同類型的 API 密鑰的選項：

如需不同類型 API 金鑰的說明，請參閱以下各節。請注意，您必須註冊付費方案才能取得完整存取金鑰。

完整存取金鑰

完整的訪問密鑰允許對 Zencoder API 發出任何類型的請求。這是您將需要提交工作給 Zencoder 並獲得輸出的關鍵。

唯讀金鑰

只讀訪問鍵可讓您進行GET僅向Zencoder API請求。這些對於只需要取得工作狀態，但實際上並未提交工作的使用者很有用。看使用只讀API密鑰想要查詢更多的信息。

整合模式鍵

整合模式鍵允許您提交工作，但不會處理這些工作。它們主要是對想要測試錯誤請求的系統整合商很有用。它們也可以用來在您購買 Zencoder 付費方案之前感受使用 API，或者在實際獲得輸出之前檢查請求。
