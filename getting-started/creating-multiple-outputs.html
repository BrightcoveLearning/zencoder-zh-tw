創建多個輸出

本主題說明如何為編碼作業創建多個輸出。

創建多個輸出

您可以通過在輸出數組中傳遞多個散列來指定多個輸出，如下所示：

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}
