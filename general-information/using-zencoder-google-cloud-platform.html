將 Zencoder 與 Google Cloud Storage 結合使用

本主題介紹了在Google Cloud Platform上使用Zencoder的設置。

Google雲端存儲設置

生成可互操作的GCS憑證

警告：對 Google Compute Engine 區域的支持已於 2017 年 5 月 30 日停止。

Google Cloud Storage可與Amazon S3 API互操作，但需要一些初始設置工作。要在Zencoder上開始使用GCS，您需要啟用互操作訪問從Google Cloud Console生成密鑰。看看生成開發人員密鑰在部分這個文件有關打開可互操作訪問和生成訪問密鑰的說明。

商店憑證

在為您的帳戶設置了一組可互操作的GCS憑據後，您就可以使用Zencoder安全地將其存儲在Zencoder憑證頁。

在下面添加憑據， 選擇谷歌云存儲。然後，填充Google Cloud Storage Access Key ID和Secret Access Key您新生成的憑據的參數。請務必點擊“用於通過gcs進行傳輸的默認憑據"-這將確保指定輸入網址的所有輸入或輸出網址gcs://協議將使用這些憑據。
