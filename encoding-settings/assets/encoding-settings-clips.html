起始截取 (_L)

開始剪輯：字符串

API版本：V1，V2

上級：輸出

有效值：時間碼 ( 時:分:SS.S )，或表示秒數的正數

兼容的工作類型：伏特

例子：

  • 00:00:10.0
  • 01：31：08.3
  • 60
  • 12.5

描述：

從特定時間開始，創建原始視頻的子剪輯。這可以是具有以下格式的時間碼時:分:SS.S，或表示開始的秒數的小數。請注意，實際開始時間可能並不完全準確，具體取決於視頻中關鍵幀的排列方式。

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "start_clip": "00:00:10.0" } ] }

也可以看看： clip_length

剪輯長度 (_L)

剪輯長度：字符串

API版本：V1，V2

上級：輸出

有效值：格式為的持續時間時:分:SS.S或代表秒數的正數

兼容的工作類型：全部

例子：

  • 00:00:10.0
  • 01：31：08.3
  • 60
  • 12.5

描述：

創建特定長度的子剪輯。這可以是格式為的持續時間時:分:SS.S，或以秒為單位表示剪輯長度的小數。可以與start_clip在視頻中間創建子剪輯。如果start_clip未提供，剪輯將從視頻的開頭開始。請注意，取決於視頻中關鍵幀的排列，實際剪輯的長度可能並不十分精確。還要注意，clip_length是一個持續時間，而不是子剪輯應在其上結束的媒體文件中的時間碼。

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "clip_length": "00:00:10.0" } ] }

也可以看看： start_clip
