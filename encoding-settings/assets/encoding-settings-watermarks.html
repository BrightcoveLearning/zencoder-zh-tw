水印

水印：數組或哈希

API版本：V1，V2

上級：輸出或者dynamic_profile_options

有效值：水印設置的數組或哈希

兼容的工作類型：全部

描述：

您可以使用我們的水印API將一個或多個水印添加到輸出視頻中。Zencoder每個輸出最多支持4個JPEG，BMP或PNG格式的水印文件。

  • 不支持GIF，32位RGB + alpha和64位RGB + alpha水印圖像。
  • 超出視頻邊緣的水印將不會被應用。如果發生這種情況，將在輸出上發出警告。
  • 使用VP6編解碼器時，還有一些其他限制。您只能將一個水印應用於視頻（如果傳遞了多個水印，則僅會使用第一個）。此外，不支持水印縮放，並且水印圖像必須是PNG文件。
  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": [ { "url": "s3://bucket/watermark_file.png", "x": "20", "y": "-10%", "width": 32, "height": 24 } ] } ] }

網址

網址：字符串

API版本：V1，V2

上級：輸出 / 水印

有效值：圖像文件的有效URL。

兼容的工作類型：全部

例子：

  • https://s3.amazonaws.com/bucket/img.png
  • ftp：// user：pass@example.com/path/to/watermark.jpg

描述：

用作浮水印的遠端影像檔案 URL。使用輸入 URL語法。支持S3，GCS，雲文件，HTTP / S，FTP和SFTP（帶有或不帶有身份驗證）。

Zencoder支持GIF，JPEG，BMP或PNG格式的水印文件。

對於透明水印，請使用具有Alpha或索引透明性的PNG。

支持的PNG格式

  • 8位RGB
  • 8位RGB + Alpha
  • 8位灰度
  • 8位索引
  • 8位索引+透明度
  • 16位RGB
  • 16位灰度

不支持的PNG格式

  • 8位灰度+ Alpha
  • 16位灰度+ Alpha
  • 16位RGB + Alpha
  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "url": "https://s3.amazonaws.com/bucket/img.png" } }]}

x

x:字符串或數字

API版本：V1，V2

上級：輸出 / 水印

默認： -10

兼容的工作類型：全部

例子：

  • -0
  • 320
  • 居中

描述：

在x軸（左/右）上將水印放置在視頻中的位置。可以是像素數目 (例如 100 或-20)、視訊寬度的百分比 (例如 25% 或-5%)，或「置中」' 將浮水印與視訊中央對齊。相對於視頻的左側使用正數，相對於視頻的右側使用負數。使用“-0"（作為字符串）鎖定到右側。

默認情況下，位置基於可見內容區域，不包括任何填充。用起源設置基於完整輸出的水印位置。

  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "x": 320 } }]}

也可以看看： ÿ和起源

寬度

寬度：數字或字符串

API版本：V1，V2

上級：輸出 / 水印

默認：縮放到高度，或原始圖像寬度。

有效值：正數或百分比

兼容的工作類型：全部

描述：

水印的寬度，表示為像素數（例如64個）或視頻寬度的百分比（例如10％）。如果高度如果提供（但不提供寬度），則水印圖像將按比例縮放。

  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "width": 100 } }]}

也可以看看：高度

默認：縮放到寬度，或原始圖像高度。

有效值：正數或百分比

  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "height": 100 } }]}

起源

原點：字符串

API版本：V1，V2

上級：輸出 / 水印

默認：內容

有效值：內容或框架

兼容的工作類型：全部

例子：框架

描述：

視頻中用於水印定位的部分。這僅影響aspect_mode為“pad"的作業。

  • '幀' 基於輸出的完整解析度，包括任何填充。
  • '內容' 基於可見內容區域的位置，不包括填充。
  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "origin": "content" } }]}

也可以看看： X， ÿ， 和Aspect_mode

不透明度

不透明度：浮動

API版本：V1，V2

上級：輸出 / 水印

默認：1.0

有效值：0.0至1.0

兼容的工作類型：全部

例子：0.5

描述：

通過將不透明度值設置為0.0（透明）和1.0（不透明）來使水印透明。

  { "input": "s3://zencodertesting/test.mov", "outputs": [ { "watermarks": { "opacity": 0.5 } }]}
