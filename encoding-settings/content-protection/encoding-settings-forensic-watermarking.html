簡介

取證水印將有助於保護您的優質內容免遭盜版和未經授權的內容共享。此功能還有助於快速識別內容洩漏的來源，以便採取措施。看法醫水印有關使用取證水印的要求的更多詳細信息。

forensic_watermark

forensic_watermark:String

API 版本： V2

上級：輸出

有效值：“甲”、“乙”

必須設置為A對於一個輸出，和B對於第二個輸出

兼容的工作類型：伏特

例子：一個

描述：

要在 Zencoder 中生成帶有取證水印的輸出，您需要為每個視頻指定 2 個輸出，一個輸出一種和乙以便來自每個輸出的片段可以在不同的序列中混合以產生一系列不同的帶水印的“視頻”以供播放（例如：AABBABBABA、ABBAABBABA、BABAABBAA 等）。

筆記：
  • 法醫水印不能添加到短於 5 分鐘的視頻中
  • segment_seconds必須設置為2否則不會添加取證水印

forensic_watermark_stub_mode

forensic_watermark_stub_mode:Boolean

API 版本： V2

上級：輸出

有效值：真假

默認值：錯誤的

兼容的工作類型：伏特

例子：真的

描述：

到用於測試集成的目的在 Zencoder 中生成帶有取證水印的輸出。當設置為true和...一起forensic_watermark，此標誌會導致創建可見水印（以驗證是否已成功添加取證水印）。

測試完成後，您將需要再次運行該作業，省略forensic_watermark_stub_mode標誌或將其設置為false , 以獲得沒有可見水印的輸出。
