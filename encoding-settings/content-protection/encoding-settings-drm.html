金鑰 _ID

key_id:String

API版本：V2

家長：產出 / 博士

有效值：32個十六進製字符（用於UUID格式的可選破折號）

兼容的工作類型：全部

例子：526d3480-48d5-11e4-916c-0800200c9a66

描述：

要使用的金鑰識別碼。所有字節均以標準（大端）順序指定。

一些 DRM 服務以格式為 GUID 的小端順序返回密鑰 ID；要以該格式指定密鑰，請參閱key_guid .

    {
      "input": "s3://zencodertesting/test.mov",
      "outputs": [
        {
          "drm": {
            "method": "cenc",
            "content_key": "25e75bb7-5c82-4213-a2d8-0796683bc409",
            "key_id": "526d3480-48d5-11e4-916c-0800200c9a66"
          }
        }
      ]
    }

也可以看看：方法 , 內容鍵， 和模式

鍵 (_L)

key_guid:String

API版本：V2

家長：產出 / 博士

有效值：32個十六進製字符（用於UUID格式的可選破折號）

兼容的工作類型：全部

例子：80346d52-d548-e411-916c-0800200c9a66

描述：

以GUID格式的little-endian順序使用的密鑰的標識符。某些DRM服務使用此格式返回密鑰ID。

如果一個key_id也未指定，這將被轉換為用作 key_id。要以標準大端二進製或 UUID 格式指定密鑰 ID，請參閱key_id .

    {
      "input": "s3://zencodertesting/test.mov",
      "outputs": [
        {
          "drm": {
            "method": "cenc",
            "content_key": "25e75bb7-5c82-4213-a2d8-0796683bc409",
            "key_guid": "80346d52-d548-e411-916c-0800200c9a66"
          }
        }
      ]
    }

輪廓

schemas:Array

API版本：V2

家長：產出 / 博士

有效值：模式數組

兼容的工作類型：全部

例子：25e75bb7-5c82-4213-a2d8-0796683bc409

描述：

使用公共加密對輸出（或一組輸出）進行加密時，有關DRM提供程序的信息將指定為一種模式，其中包括模式類型和特定於提供程序的信息。

通常，僅在播放列表輸出上為多比特率編碼作業指定此參數，但也可以為CENC編碼的輸出指定此參數，以確保輸出可單獨播放。

    {
      "input": "s3://zencodertesting/test.mov",
      "outputs": [
        {
          "drm": {
            "method": "cenc",
            "content_key": "5897a089-98dc-4d43-96f2-d6857d69b196",
            "key_id": "526d3480-48d5-11e4-916c-0800200c9a66",
            "schemas": [
              {
                "type": "marlin"
              },
              {
                "type": "playready",
                "license_acquisition_url": "https://license.server.example.com/license/123456789"
              },
              {
                "type": "widevine",
                "license_acquisition_url": "https://widevine.example.com/license_server/"
              }
            ]
          }
        }
      ]
    }

也可以看看：方法 , 內容鍵 , key_id， 和key_guid
