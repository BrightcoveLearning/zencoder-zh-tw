編碼建議

AV1是開放媒體視頻聯盟的一種開放的，免版稅的下一代視頻編碼格式。與HEVC，VP9和H.264編解碼器相比，它可以以低得多的比特率提供卓越的質量。

見AV1指南有關更多信息，請特別注意局限性獲得當前支持。

除了在此頁面上找到的設置外，您可能還希望包括其他常規視頻設置，但請注意，以下設置是不是目前支持：

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

另請注意分辨率設定（是否使用size或者width和height）寬度和高度必須是8的倍數。

視訊轉碼器層級

video_codec_level:String

API版本：V2

上級：輸出

有效值：

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

兼容的工作類型：全部

例子：4.1

描述：

限制比特率和編碼樹單元（CTU）（AV1）。

請注意，此設置可用於具有不同允許值的其他編解碼器。看視頻編碼設置。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

視訊轉碼器輪廓

video_codec_profile:String

API版本：V2

上級：輸出

有效值：main，high，professional

默認：main

兼容的工作類型：全部

例子：4.1

描述：

設定編碼設定檔。

請注意，此設置可用於具有不同允許值的其他編解碼器。看視頻編碼設置。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}
