格式

format:String

API版本：V1，V2

上級：輸出

默認：取決於輸出檔案名稱，然後是視訊或音訊轉碼器。否則：mp4（用於標準輸出）; ts（用於分段輸出）。

有效值：3GP，AAC，AC3，EC3，FLV，M4F，MJ2，MKV，MP3，MP4，MXF，OGG，TS，WebM和WMV

兼容的工作類型：全部

例子： Webm

描述：

要使用的輸出容器格式。

請注意，某些容器具有多個有效的文件擴展名。通常，名稱“f4v"，“ogv"和“m4v"用於音頻/視頻文件，“f4a"，“m4a"和“oga"用於僅音頻文件，但這不是強制性的。同樣，“f4v"和“f4a"通常用於設計在Flash Video中播放的mp4內容，但這只是一個約定。

兼容性：

  • mp4與h264（默認），hevc，mpeg4， 和AV1視頻和AAC（默認），MP3，AC3和EAC3音頻。有效值和文件擴展名：mp4，m4a，m4v，f4v，f4a，m4b，m4r，f4b
  • m4f與h264和hevc視頻和AAC，AC3和EAC3音頻。
  • ogg與theora視頻和vorbis音頻。有效值和文件擴展名：ogg，oga，ogv，ogx
  • webm與vp8（默認），vp9和AV1視頻和vorbis音頻兼容。有效值和文件擴展名：webm
  • 3gp與mpeg4（默認）和h264視頻，AAC（默認）和mp3音頻。有效值和文件擴展名：3gp，3gp2、3g2、3gpp，3gpp2。請注意，3gp格式也可以使用mp4文件擴展名。
  • flv與vp6（默認）兼容，並且h264視頻，AAC（默認）和mp3音頻。有效值和文件擴展名：flv
  • ts與h264視頻和aac（默認），mp3或ac3音頻。有效值和文件擴展名：ts
  • mkv與h264視頻和AAC（默認），MP3，AC3或EAC3音頻。有效值和文件擴展名：mkv
  • wmv與wmv視頻和wma（默認）或mp3音頻兼容。有效值和文件擴展名：wmv，wma
  • aac與aac音頻兼容，沒有視頻。有效值和文件擴展名：aac
  • mp3與mp3音頻兼容，沒有視頻。有效值和文件擴展名：mp3
  • ac3與ac3音頻兼容，沒有視頻。有效值和文件擴展名：ac3
  • ec3與eac3音頻兼容，沒有視頻。有效值和文件擴展名：ec3
  • mj2與jp2視頻和AAC音頻。有效值和文件擴展名：mj2，mjp2。
  • mxf與h264視頻和pcm音頻。有效值和文件擴展名：mxf。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "format": "webm"
    }
  ]
}

視訊編解碼器

video_codec:String

API版本：V1，V2

上級：輸出

默認：由格式，配置文件或audio_codec。h264如果沒有提供。

有效值：h264，hevc，av1，jp2，mpeg4，theora，vp6，vp8，vp9和wmv

兼容的工作類型：全部

例子： vp8

描述：

要使用的輸出視頻編解碼器。Zencoder當前支持以下編解碼器：h264（默認），hevc，jp2，theora，vp6，vp8，vp9，mpeg4， 和wmv。

兼容性：

  • h264與...兼容aac（默認）和mp3音頻和mp4（默認），m4f，flv， 或者3gp格式
  • hevc與aac（默認）或mp3音頻以及mp4或m4f格式兼容。
  • vp9與vorbis音頻和webm格式兼容
  • vp8與vorbis音頻和webm格式兼容
  • vp6與mp3（默認）或aac音頻以及flv格式兼容
  • theora與vorbis音頻和ogg格式兼容
  • mpeg4與aac（默認）和mp3音頻以及mp4（默認）或3gp格式兼容
  • wmv與wma（默認）或mp3音頻以及wmv格式兼容
  • jp2與pcm音頻和mj2格式兼容。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec": "vp8"
    }
  ]
}

音訊編解碼器

audio_codec:String

API版本：V1，V2

上級：輸出

默認：由格式，配置文件或video_codec。 AAC（如果未提供）。

有效值： aac，ac3，amr，av1，eac3，mp3，pcm，vorbis和wma

兼容的工作類型：全部

例子： mp3

描述：

要使用的音訊轉碼器。默認編解碼器取決於所選的視頻編解碼器和格式。

請注意，MP4容器中的MP3音頻與H.264視頻內容一起有效，但不能在Quicktime中播放。

兼容性：

  • AAC與h264視頻（mp4，m4f，3gp或flv格式），mpeg4視頻（mp4或3gp格式）和vp6（flv格式）
  • vorbis是兼容的vp8視頻（webm格式），並且theora視頻（ogg格式）
  • wma與wmv視頻和wmv格式兼容
  • mp3與h264視頻（mp4、3gp或flv格式）和wmv視頻（wmv格式）。 mp3，h264，而mp4將無法在Quicktime中播放
  • pcm與h264視頻和mxf格式。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "audio_codec": "mp3"
    }
  ]
}
