暗示

hint:Boolean

API版本：V1，V2

上級：輸出

默認：錯誤的

有效值：對或錯

兼容的工作類型：全部

例子：真的

描述：

打開用於RTP / RTSP的MP4文件的提示。默認值mtu_size是1450字節；使用mtu_size更改此選項。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

數量 _ 大小

mtu_size:Integer

API版本：V1，V2

上級：輸出

默認：1450

有效值：100到5000之間的整數

兼容的工作類型：全部

例子：2000

描述：

用於MP4提示的MTU大小（以字節為單位）。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}
