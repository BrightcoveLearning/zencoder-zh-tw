迴轉

rotate:Integer

API版本：V1，V2

上級：輸出

默認：自動檢測

有效值：0、90、180、270。如果要自動旋轉，請保持未設置狀態。

兼容的工作類型：伏特

例子：180

描述：

明確旋轉影片。默認情況下，Zencoder將自動旋轉旋轉的內容（包括旋轉的iPhone視頻），但您也可以手動旋轉電影。如果將“旋轉"設置為0，我們將不會自動旋轉，因此如果要自動旋轉，請不要設置此選項。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "rotate": 180
    }
  ]
}

去交錯

deinterlace:String

API版本：V1，V2

上級：輸出

默認：探測

有效值：開啟，關閉，檢測

兼容的工作類型：伏特

例子：上

描述：

確定是否應用去隔行濾鏡。默認值為“檢測" –如果檢測到輸入文件為隔行掃描，則將對其進行隔行掃描。將此設置為“開"以強制去隔行（如果輸入不是隔行的，這會降低質量），或設置為“關"以避免反隔行。

請注意，Zencoder基於文件元數據進行此確定。如果隔行掃描的內容是使用不會對內容進行隔行掃描或將其標記為隔行掃描的編碼器進行重新編碼的，則我們可能無法在“檢測"模式下自動對隔行掃描內容進行隔行掃描。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "deinterlace": "on"
    }
  ]
}

表示

denoise:String

API版本：V1，V2

上級：輸出

有效值：弱，中，強或最強

兼容的工作類型：伏特

例子：虛弱的

描述：

將降噪濾鏡應用於視頻。

有效選項為“弱"，“中"，“強"和“最強"。通常，根據降噪級別，比特率和源視頻，將文件大小減小5％-20％。任何降噪都會使編碼速度降低約20％。

高於 '弱的值' 應該仔細使用，儘管動畫有時可以處理高度的去噪。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "denoise": "weak"
    }
  ]
}

自動層級

autolevel:Boolean

API版本：V1，V2

上級：輸出

默認：錯誤的

有效值：對或錯

兼容的工作類型：伏特

例子：真的

描述：

對視頻應用顏色自動級別過濾器。請注意，此濾鏡可能會在某些視頻中引起意外的顏色問題，因此請小心使用。

VP6 video_codec不支持自動級別選項。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "autolevel": true
    }
  ]
}
