布萊特灣持續監控所有布賴特灣服務和應用的狀態。如果服務中斷，系統狀態頁面會張貼一則備註。所有定期的維修通知也將在此頁面上張貼。

要訪問系統狀態頁面，請訪問https://status.brightcove.com或從導航標題中，單擊支持圖標 (） 進而 系統狀況 .

所有張貼的通知都會包含下列資訊：

  • 事件狀態
    • 運作中 (綠色)
    • 通知 (藍色 — 系統目前正在運作，但受到延遲或間歇性中斷影響)
    • 部分服務中斷 (黃色)
    • 全面服務中斷 (紅色)
  • 元件-服務或受影響區域
  • 位置
    • 亞太地區
    • 美洲
    • 歐美
    • 日本
  • 目前的狀態和狀態。狀態包括：
    • 調查-工程師正在研究這個問題
    • 確定-工程師已經確定了這個問題，並正在尋找糾正
    • 監控-問題解決和工程監控，確保一切健康
    • 已解決-問題已解決

所有發佈的訊息都以英文顯示，使用的時間為東部標準時間 (GMT-5)。

檢視系統狀態歷史記錄

要查看系統狀態歷史記錄，請單擊歷史頁面底部的鏈接。

訂閱系統狀態變更

您可以註冊，以便在系統狀態變更時收到通知。本節的組織方式如下：

  • 訂閱電子郵件
  • 訂閱webhook
  • 管理您的訂閱

訂閱電子郵件

要通過電子郵件接收通知，請單擊訂閱按鈕，輸入您的電子郵件地址，然後單擊訂閱。

訂閱網頁鉤

網絡鉤是用 HTTP POST 製作的定義回調。您可以指定一個 URL，當有狀態更新時，回調將發布 JSON 對象。

要註冊以在系統狀態更改時收到通知，請選擇訂閱按鈕。然後輸入一個 URL 以接收 JSON 承載以及您的電子郵件地址。

管理您的訂閱

訂閱之後，您會收到一封電子郵件，其中包含管理訂閱頁面的連結，讓您管理訂閱。您可以選擇您感興趣接收通知的元件/產品/地區。進行選擇，然後單擊保存訂閱。

要取消訂閱，請點擊退訂鏈接位於“管理訂閱"頁面底部。

在Slack中獲取通知

Slack有自己的webhook系統，可用於連接狀態更新。諮詢鬆弛文檔以獲取詳細信息。
