Brightcove 提供各種支援方案，以滿足客戶的需求。有關這些程序的信息，請檢查Brightcove支持部分我們的網站。

其中一個支援選項是支援入口網站，可用來開啟與 Brightcove 支援的案例

建立新的支援帳戶

您必須先建立帳戶，才能使用支援入口網站建立案例。若要建立新的帳戶，請依照下列步驟執行：

  1. 使用下列其中一種方法存取支援入口網站：
    • 在應用程序標題中，單擊支持>聯繫支持
    • 去https://supportportal.brightcove.com
  2. 點擊報名。
  3. 輸入您的名，姓，電子郵件和密碼。點擊提交。
  4. 確認您被重新導向至 Brightcove 支援入口網站首頁。

開啟支援案例

若要使用支援入口網站與 Brightcove 支援開啟案例，請依照下列步驟執行：

  1. 登錄到Brightcove支持門戶，網址為： https://supportportal.brightcove.com。
  2. 點擊提交案例在頁面標題中。
  3. 輸入案例資訊：
    • 主題 -案件名稱
    • 描述 -問題，問題或問題的描述
    • 優先事項
      • P1 -嚴重-丟失了Brightcove服務，或嚴重喪失了實時製作播放器的大量發布可用性
      • P2 -重要-Brightcove Service可以運行，但是存在與現場製作播放器特定的預期發布功能有關的阻止問題（即與上載，控制台活動，特定播放器有關）。
      • P3 -正常-未定義為“嚴重"或“重要"的請求（包括帳戶/用戶修改請求）
    • 產品 -與支持請求有關的Brightcove應用程序
    • Brightcove帳戶 -支持請求所涉及的帳戶名
    • 網址 -可以發現問題的頁面網址（如果有）
    • 附加CC -在默認抄送列表的頂部，您可以決定添加協作者的其他電子郵件地址，以告知案件進展情況
  4. 點擊提交。將顯示個案詳細資料。
  5. 點擊寫一個新的評論...鏈接以添加其他評論或上傳屏幕截圖。重現行為/問題的問題和螢幕擷取畫面的詳細步驟對 Brightcove 支援一律有幫助，可加快解決個案所需的時間。

檢視您的支援案例

注意：如果您的支持門戶配置文件是行政，您將看到該帳戶的所有案例（其他帳戶用戶創建案例時您不會收到通知）。如果你是一個標準或自行註冊的用戶，您只會看到您打開的案例。< /在旁邊

您可以在支援入口網站輕鬆檢視所有支援案例。這案例狀態將是以下之一：

  • 新的 -案例剛剛創建，正在等待Brightcove審核
  • 打開 -案件正在進行中
  • 等待你的回复 -Brightcove支持需要更多信息或已提供解決方案，並且Brightcove正在等待響應
  • 關閉 -案件已解決
  • 封閉等待 -由於發布商未回應，此案已結案；如果發布商聯繫Brightcove，將重新開箱
請注意，已被支持人員刪除的重複案例不會出現在支持門戶中。

若要使用 Brightcove 支援檢視您的案例，請依照下列步驟執行：

  1. 登錄到Brightcove支持門戶，網址為： https://supportportal.brightcove.com。
  2. 點擊支持案例在頁面標題中。的列表開箱將顯示。
  3. 點擊病歷鏈接以查看已結案的列表。
  4. 點擊一個案件編號查看案例詳細信息，包括Brightcove支持人員的評論。

按一下欄標題會依該欄排序清單。

更新支援案例

建立案例後，您可以採取以下幾個動作，讓案例保持最新狀態：

  • 新增意見與我們的支援團隊溝通
  • 上傳或刪除附件的詳細資訊，以協助我們盡快找出問題
  • 如果您需要更多關注某個案例，或變得比最初預期的緊急情況，請更新案例優先順序
  • 更新 CC 清單以新增更多觀察者至案例
  • 問題解決後要求關閉案例

更新您的預設副本清單

預設的副本清單是每個帳戶的電子郵件清單，通知帳戶的所有案例活動。如果要更新清單，請依照下列步驟執行：

  1. 轉到右上角，點擊用戶圖標。
  2. 點擊我的帳戶。
  3. 輸入新的電子郵件地址或刪除現有的地址帳戶默認抄送列表。
  4. 點擊更新默認抄送列表保存更改。

檢視布萊特灣系統狀態頁面

布萊特灣持續監控所有布賴特灣服務的狀態。如果服務中斷，系統狀態頁面會張貼一則備註。所有定期的維修通知也將被張貼。

要訪問“系統狀態"頁面，請登錄支持門戶，然後單擊頁面標題中的“系統狀態"或轉到status.brightcove.com。

要了解有關“系統狀態"頁面的更多信息，請參閱查看Brightcove系統狀態頁面。
