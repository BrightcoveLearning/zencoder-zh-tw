AVC內部概述

AVC-Intra定義兩個預設，AVC-Intra 50和AVC-Intra 100。AVC-Intra 50的編碼速率為50mbit / s，而AVC-Intra 100的編碼速率為100mbit / s。使用video_codec_preset參數定義AVCi輸出。有關創建AS-11輸出的信息，請查閱文檔這裡。

  • video_codec_preset：設置輸出的AVC-Intra預設。值：avci_50，avci_100。

範例設定

示例AVC-Intra 50請求

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

示例AVC-Intra 100請求

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}
