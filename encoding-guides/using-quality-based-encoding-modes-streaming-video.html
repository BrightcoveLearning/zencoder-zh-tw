簡介

在對視頻輸出進行流編碼時，我們不建議使用質量單獨設置（這使用CRF編碼模式）。如果您更喜歡使用以質量為目標的模式，那麼我們強烈建議您使用VBV設置來限制比特率的變化，以符合流格式規範。

VBV設置

  • coder_bitrate_cap
  • coder_buffer_size

其他建議考慮的設置

  • video_reference_frames
  • video_bframes

範例

這是具有VBV限制的以質量為目標的編碼的示例。

{
    "type":"segmented",
    "width":640,
    "height":360,
    "quality":3,
    "video_codec_profile":"main",
    "video_reference_frames": 4,
    "video_bframes": 3,
    "decoder_bitrate_cap": 1440,
    "decoder_buffer_size": 1920,
    "keyframe_rate": 0.5,
    "max_frame_rate": 30,
    "audio_quality":5,
    "segment_seconds":10
}
